*状态码说明*

|===
| http状态码 | 简介

| 400
| 请求参数有误

| 401
| 用户验证不通过

| 403
| 资源权限校验不通过（用户校验通过，url没有权限）

| 404
| 资源未发现

| 405
| http方法权限校验不通过（用户校验通过，url权限校验通过，操作权限（put,delete）校验不通过）

| 500
| 服务器内部出错

|===
